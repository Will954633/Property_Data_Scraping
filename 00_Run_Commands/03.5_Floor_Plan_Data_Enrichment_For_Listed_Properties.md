## Key Features

✅ **Comprehensive Data Extraction:**
- Total floor area and land area with confidence levels
- Room-by-room dimensions (length, width, area)

- Parking information

✅ **Window Orientation Detection:**
- Searches for compass rose or north arrow on floor plans
- Checks for sun symbols or orientation markers

✅ **Standardized Output:**
- Consistent JSON structure across all properties

- Orientation analysis by room (north/south/east/west facing)
- Buyer insights and lifestyle suitability
